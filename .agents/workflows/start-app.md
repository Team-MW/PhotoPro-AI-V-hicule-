description: Démarrer l'application PhotoPro (Interface Web + API)
---

// turbo-all
1. Dans le premier terminal, activer l'environnement et lancer l'API :
```bash
source venv/bin/activate && python api.py
```

2. Dans un second terminal, lancer l'interface Web (Vite) :
```bash
cd frontend && npm run dev -- --port 3000
```

3. Ouvrez votre navigateur sur : http://localhost:3000
